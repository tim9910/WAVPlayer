# WAV 音效檔播放器

## 概述
**WAV 音效檔播放器** 是一個簡單的 **Windows Forms** 應用程式，用於播放和控制 .wav 格式的音效檔案。用戶可以選擇音效檔案並進行播放、重複播放、停止播放及結束程式等操作。

## 開發環境

- **語言**：C#
- **IDE**：Visual Studio 2022
- **平台**：Windows

## 功能

|      功能       |   圖示  |     功能說明           |
|-----------------|---------|------------------------|
|**選擇音效檔案** | ![選擇音效檔案](https://github.com/tim9910/tim9910.github.io/blob/main/images/wavplayer/add.png) |使用者可透過檔案選擇對話框選擇 .wav 格式的音效檔案。  |
|**播放音效**     | ![播放](https://github.com/tim9910/tim9910.github.io/blob/main/images/wavplayer/play.png) |可播放選擇的音效檔案  |
|**重複播放**     | ![重複播放](https://github.com/tim9910/tim9910.github.io/blob/main/images/wavplayer/repeat.png) |可以讓音效在播放結束後自動重新播放  |
|**停止播放**     | ![停止播放](https://github.com/tim9910/tim9910.github.io/blob/main/images/wavplayer/stop.png) |可停止目前播放的音效  |
|**結束程式**     | ![結束程式](https://github.com/tim9910/tim9910.github.io/blob/main/images/wavplayer/logout.png) |可選擇結束應用程式  |
